# Launch in 30 Days

AI-powered business launch system — from zero to open doors in 30 days.

## Setup

1. Replace `YOUR_GUMROAD_LINK` in `index.html` with your actual Gumroad product URL
2. Push to GitHub
3. Connect to Vercel for instant deployment

## Files

- `index.html` — Landing page (sales page)
- `app.html` — Main app (what buyers get access to)
- `vercel.json` — Vercel routing config

## Monetization

- Price: $67 one-time on Gumroad
- After purchase: Gumroad delivers the app URL to the buyer
